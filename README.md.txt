Student Management System
A Python-based Student Management System designed to manage student records efficiently using file handling and database concepts. The application allows users to add, update, delete, search, and display student information through a simple and user-friendly interface. This project helps in understanding CRUD operations, data management, and backend logic implementation using Python.

Features
Add new student records
View all student details
Search students by ID or name
Update existing student information
Delete student records
Store and manage data efficiently
Simple command-line interface for easy interaction

Technologies Used
Python
File Handling
Object-Oriented Programming (OOP)

How It Works
The system allows users to manage student information such as:
Student ID
Name
Marks/Grade
Users can perform CRUD operations:
Create
Read
Update
Delete
